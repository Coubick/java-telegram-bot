# Телеграм-бот на Java
## Технологии:
### Java 21
### Сборка: maven
### SpringBoot 4.0.1
### TelegramBots 6.9.7.1
### В основе архитектуры - паттерн Команда
### СУБД: PostgreSQL


## Функционал:
### 1) есть некоторые капитал, на который каждые сутки поступают монетки
### 2) можно покупать прокруты автомата на эти монетки. как, сколько денег, сколько прокрутов - решать пользователю.
### 3) какие команды планируется реализовать:
   &emsp; `/start` запустить бота (✅)
   
   &emsp; `/register <nickname>` - регистрация, nickanme - имя пользователя в игре, которое он вводит сам (✅)
   
   &emsp; `/my_stat` — вывод статистики пользователя (✅)
   
   &emsp; `/rating` — рейтинг всех пользователей по выигрышам (✅)
   
   &emsp; `/dep <money> <spins amount>` — поставить ставку (money) на количество слотов (spins amount). Пример вызова: `dep 100 10`. В таком случае стоимость одного слота = 10  (✅)
   
   &emsp; `/rules` - получить правила игры (✅)

!! Весь функционал и правила пока что ОЧЕНЬ примерные.
