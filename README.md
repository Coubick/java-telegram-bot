# Телеграм-бот на Java
## Технологии:
### Java 21
### Сборка: maven
### SpringBoot 4.0.1
### TelegramBots 6.9.7.1
### В основе архитектуры - паттерн Команда
*список будет пополняться*

## Функционал:
### 1) есть некоторые капитал, на который каждые сутки поступают монетки
### 2) можно покупать прокруты автомата на эти монетки. как, сколько денег, сколько прокрутов - решать пользователю.
### 3) какие команды планируется реализовать:
   &emsp; `/entrance` - регистрация, надо вызвать, получить никнейм и начать игру (если никнейм не понравится, можно переделать)
   
   &emsp; `/regenerate` - создать новый никнейм
   
   &emsp; `/my_stat` — вывод статы по депам, выигрышам и доступным деньгам на счете и полный капитал
   
   &emsp; `/rating` — рейтинг всех пользователей с выигрышами
   
   &emsp; `/dep <money>` — занести деньги в казик из своего капитала и крутануть слот
   
   &emsp; `/rules` - правила игры

!! Весь функционал пока что примерный, т.к. я не знаком с возможностями библиотеки
